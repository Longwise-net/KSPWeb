# KSPWeb

A minimalist web browser designed to be a modern yet classic web experience. Built with Turbowarp, KSPWeb offers a lightweight alternative to traditional web browsers.

## Current Features

- **Search Bar** - Quickly search and navigate to websites
- **IFRAME Integration** - Seamlessly embed and display web content
- **Sidebar Navigation** - Clean, organized sidebar for easy access and extensibility

## About Turbowarp

KSPWeb is programmed using [Turbowarp](https://turbowarp.org/), a powerful modification of Scratch that enables the creation of sophisticated applications beyond traditional visual programming.

## Project Status

This project is in active development with a focus on building a minimalist browser experience. The sidebar is designed for future expansion with additional navigation and utility features.

## Getting Started

To use KSPWeb, simply load the project and start exploring the web with the search bar. The sidebar provides quick access to features and can be expanded as the project develops.
